# PPSU Online — Admissions Frontend

Next.js 15 + TypeScript + Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Structure

```
app/
  layout.tsx        # Root layout (fonts, metadata)
  page.tsx          # Home page — assembles all sections
  globals.css       # Global styles + Tailwind base

components/
  TopBar.tsx
  Header.tsx
  HeroBanner.tsx
  StatsBar.tsx
  WhyPPSU.tsx
  Placement.tsx
  Courses.tsx
  Scholarships.tsx
  InternationalCollab.tsx
  CampusGallery.tsx
  CampusVideos.tsx
  Testimonials.tsx
  FAQ.tsx
  Footer.tsx
  SidebarApply.tsx
  StickyPhone.tsx

public/             # Static assets (images, icons)
```
